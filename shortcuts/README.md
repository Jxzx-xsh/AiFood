# iOS 快捷指令配置指南

## 快捷指令名称：录入食物

### 配置步骤

1. 打开 iPhone【快捷指令】App
2. 点击右上角 `+` 新建快捷指令
3. 命名为 **"录入食物"**

### 操作流程（按顺序添加）

#### 第 1 步：拍摄照片
- 操作：**拍摄照片/视频**（或选择 **从照片中选取** 允许选相册图片）
- 关闭"显示相机预览"以加快速度

#### 第 2 步：Base64 编码
- 操作：**Base64 编码**
- 输入：上一步的 **照片**

#### 第 3 步：获取当前日期
- 操作：**日期**
- 格式：选择 **ISO 8601**（自定义格式: `yyyy-MM-dd'T'HH:mm:ssZ`）

#### 第 4 步：构建请求体
- 操作：**文字**
- 内容：
```
{"family_id":"00000000-0000-0000-0000-000000000001","image_base64":"data:image/jpeg;base64,【Base64编码结果】","shoot_time":"【日期变量】"}
```

#### 第 5 步：发送网络请求
- 操作：**获取 URL 内容**
- URL：`http://你的服务器IP:3000/api/v1/ingest/photo`
- 方法：`POST`
- 头部：
  - `Content-Type`: `application/json`
  - `X-API-Key`: `你的API密钥`
- 请求体：选择 **文件**，输入上一步的 **文字** 结果

#### 第 6 步：解析结果
- 操作：**从输入获取字典值**
- 取 `data.total_saved` 字段

#### 第 7 步：显示通知
- 操作：**显示通知**
- 标题：食物录入完成
- 内容：`成功识别并保存了 【字典值】 项食物`

---

### 进阶配置

#### 添加到主屏幕
- 在快捷指令详情中，点击"添加到主屏幕"
- 选一个食物相关的图标（如 🍎）

#### Siri 语音触发
- 设置 → Siri → 快捷指令
- 添加语音短语，如 **"记录食物"**

#### 自动化建议
- 可设置基于 NFC 标签触发（贴在冰箱上）
- 每次开冰箱门时自动打开拍照界面

---

### 错误处理
如果请求失败，快捷指令会显示错误信息。常见问题：
- **连接失败**：检查手机与服务器是否在同一网络
- **401 错误**：API Key 配置错误
- **502 错误**：LMStudio 服务未启动
- **超时**：视觉模型推理可能需要 30-60 秒，请耐心等待
